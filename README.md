# tic-tac-toe-ai-frontend
AIと対戦できる三目並べのfrontendの実装です。Reactで記述されています。

## セットアップ
``` shell
npm install
```

## 実行方法
``` shell
npm start
```

[`http://localhost:3000/`](http://localhost:3000/)からアクセスできます。

backendの実装は以下のリポジトリにあります。
